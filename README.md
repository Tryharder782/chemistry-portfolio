# Acids & Bases Portfolio Build

This repository contains an isolated portfolio build of the **Acids & Bases** simulation module.

## Scope

- Includes only the Acids & Bases flows:
  - Introduction
  - Buffers
  - Titration
  - Quizzes
  - Experiment History
- Routing is restricted to `/acids/*`.
- App entry redirects `/` to `/acids/chapters`.

## Run

```bash
npm ci
npm start
```

## Production build

```bash
npm run build
```
